# Minishell

## Description
Minishell is a collaborative project at 42 where we developed a simplified shell inspired by Bash. 
The goal was to understand system calls, process creation, and signal handling in a Unix environment.

Key features include:
- Command execution using absolute and relative paths.
- Redirections (`>`, `<`, `>>`, `<<`).
- Pipes (`|`) to chain commands.
- Management of environment variables and expansion of `$`.
- Implementation of built-in commands like `echo`, `cd`, `pwd`, and `export`.

## Instructions
To compile program use 'make' then run minishell using: ./minishell

<pre>
make
./minishell
</pre>
